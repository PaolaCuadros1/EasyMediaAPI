# Easy Media API

## Installation

### Considerations

1. You must set the PATH to **firebase.json** file into `.env` file.

```env
GOOGLE_APPLICATION_CREDENTIALS="D:\AngularProjects\...\EasyMediaAPI\firebase.json"
```

### Commands

```shell
cd EasyMediAPI
npm install
```

## Run

```shell
PORT=2500 npm run dev
```
